The optgroup disabled attribute



Definition and Usage

The disabled attribute is a boolean attribute. When present, it specifies that an option-group should be disabled. A disabled option-group is unusable and un-clickable. The disabled attribute can be set to keep a user from selecting the option-group until some other condition has been met (like selecting a checkbox, etc.). Then, a JavaScript is required to remove the disabled value, and make the option-group selectable.

Syntax: <optgroup disabled>
