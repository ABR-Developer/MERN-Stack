The select size attribute

The size attribute specifies the number of visible options in a drop-down list:



Definition and Usage

The size attribute specifies the number of visible options in a drop-down list. If the value of the size attribute is greater than 1, but lower than the total number of options in the list, the browser will add a scroll bar to indicate that there are more options to view. Note: In Chrome and Safari, this attribute may not work as expected for size="2" and size="3".

Syntax: <select size="number">
