The select disabled attribute

The disabled attribute specifies that the drop-down list should be disable:



Definition and Usage

The disabled attribute is a boolean attribute. When present, it specifies that the drop-down list should be disabled. A disabled drop-down list is unusable and un-clickable. The disabled attribute can be set to keep a user from using the drop-down list until some other condition has been met (like selecting a checkbox, etc.). Then, a JavaScript can remove the disabled value, and make the drop-down list usable.

Syntax: <select disabled>
