Definition and Usage

The name attribute specifies the name for a <button> element. The name attribute is used to reference form-data after the form has been submitted, or to reference the element in a JavaScript. Tip: Several <button> elements can share the same name. This allows you to have several buttons with equal names, which can submit different values when used in a form.

Syntax: <button name="name">
