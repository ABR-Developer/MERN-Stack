The button popovertargetaction Attribute

Hello

Click the button and it will show the popover element.

Definition and Usage

The popovertargetaction attribute allows you to define what happens when you click the button. You can choose between the values "show", "hide", and "toggle". If the popovertargetaction attribute is not specified, the default "toggle" value will be used.

Syntax: <button popovertarget="element_id popovertargetaction="hide|show|toggle"">

Values

Value Description
hide The popover element is hidden when you click the button Try it »
show The popover element is showed when you click the button Try it »
toggle Default value. The popover element is toggled between hidding and showing when you click the button Try it »
