The input popovertargetaction Attribute

Hello

Click the button and it will show the popover element.

Definition and Usage

The popovertargetaction attribute allows you to define what happens when you click the button. You can choose between the values "show", "hide", and "toggle". The popovertargetaction only works when type="button". If the popovertargetaction attribute is not specified, the default "toggle" value will be used.

Syntax: <input type="button" popovertarget="element_id popovertargetaction="hide|show|toggle"">
