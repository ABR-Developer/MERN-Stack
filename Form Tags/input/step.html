The input step attribute

Definition and Usage

The step attribute specifies the interval between legal numbers in an <input> element. Example: if step="3", legal numbers could be -3, 0, 3, 6, etc. Tip: The step attribute can be used together with the max and min attributes to create a range of legal values. Note: The step attribute works with the following input types: number, range, date, datetime-local, month, time and week.
