The label form attribute

The label for the first radio button is outside the form, but still a part of the form. Try to click the "HTML" text label to select the "HTML" radio button.

 
   
   

Definition and Usage

The form attribute specifies the form the label belongs to. The value of this attribute must be equal to the id attribute of a <form> element in the same document.

Syntax: <label form="form_id">
