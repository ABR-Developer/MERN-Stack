The script element

Definition and Usage

        The <script> tag is used to embed a client-side script (JavaScript).
        The <script> element either contains scripting statements, or it points
        to an external script file through the src attribute.
        Common uses for JavaScript are image manipulation, form validation, and dynamic changes of content.

Tips and Notes

        Tip: Also look at the <noscript> element for users that have disabled
        scripts in their browser, or have a browser that doesn't support client-side scripting.
