It's worth noting that while various MIME types can be used, in practice, 
        JavaScript is predominantly used on the web, and the default application/javascript
        or simply omitting the type attribute is common.
  • Using application/javascript (default): type="application/javascript" In modern HTML5, the type attribute is often omitted, and the default assumed type is JavaScript.
  • Using module: type="module" The module type is used when writing JavaScript code as ES6 modules. The src attribute will be treated as a module specifier.
